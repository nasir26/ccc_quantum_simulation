# CCC Quantum Simulation

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Quantum circuit simulation experiments and prototyping scripts at C-DAC Noida.

**Author**: Nasir Ali — Centre for Development of Advanced Computing (C-DAC), Noida

## Overview

Prototype quantum simulation scripts exploring various algorithms and circuit
structures before integration into the main FPGA simulation framework.

## Contents

- Circuit prototyping (QFT, Grover, VQE)
- Noise model experiments
- Algorithm benchmarking scripts

## Requirements

```bash
pip install qiskit numpy scipy matplotlib
```
